> Physical drive imaging and formatting utility for Linux, written in Python.

## Requirements
- `base-devel`
- `git`

## Installation
```bash
git clone https://github.com/Saber0324/lufus-pkgbuild
cd lufus-pkgbuild
makepkg -si
```

Now available in the AUR as "lufus". 
Download with your preferred AUR helper. 

```bash
yay -S lufus
paru -S lufus
```
